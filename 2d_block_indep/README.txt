This zip file contains all code necessary to train 2D Block and 2D Indep problem. 
The folder structure has already been set up. 
The data folder needs to be filled with data generated by the data generation script. 
Running learn_*.py will start training on the respective architecture. 
Checkpoint files will be saved to ckpt_* folder. 
Please email yilun@cs.duke.edu for questions.
